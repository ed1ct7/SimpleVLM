# solution/ — код и артефакты решения

Сюда агент-сессии складывают результаты работы. Это будущий публикуемый репозиторий.

## Ожидаемая структура (создаётся по мере выполнения)

```
solution/
├── env/            # 01: инструкция/скрипты окружения, requirements, проверка GPU
├── data/           # 02: скрипты загрузки LLaVA-Instruct-ru + COCO 2014, лоадер
├── train/          # 03–04: код обучения (стадии 1–2, конфиги QLoRA)
├── eval/           # 05: скрипты оценки GQA-ru + MMBench-ru
├── model/          # обученные адаптеры/проектор или ссылка на HF Hub
├── results/        # 05: таблицы метрик
├── README.md       # 06: описание модели + инструкция запуска
└── SOLUTION.md     # 06: подробное описание решения (обязательное требование №4)
```

## Правила
- Каждая задача пишет в свою подпапку.
- Большие веса/данные **не коммитить** — публиковать на HF Hub / давать ссылку.
- Добавить `.gitignore` (данные COCO, чекпойнты, `__pycache__`, `.venv`) на этапе 02.
